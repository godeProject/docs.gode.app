# docs.gode.app

gode.js documentation website.

## Running locally

Install dependencies (Globally recommended)

With npm

```bash
$ npm i docsify-cli -g
```

or with yarn

```bash
$ yarn global add docsify-cli
```

Then run docsify serve to serve with hot reload

```bash
$ docsify serve ./docs
```
